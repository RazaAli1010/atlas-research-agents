# Atlas frontend

React 19 + TypeScript + Vite + Tailwind CSS v4 SPA. See the root `README.md` for setup
and the project `CLAUDE.md` for the full spec.

```bash
npm install
npm run dev        # http://localhost:5173
npm run lint       # eslint (flat config)
npm run typecheck  # tsc --noEmit
npm run test       # vitest
npm run build      # tsc -b && vite build
```
